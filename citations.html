Code & Tutorials

Scroll-Triggered Animations (IntersectionObserver)

Used for:

The fade-up reveal effect on cards and panels as the user scrolls down the page

Source:

MDN Web Docs — Intersection Observer API
https://developer.mozilla.org/en-US/docs/Web/API/Intersection_Observer_API

Official browser documentation explaining how to detect when an element scrolls into view, used as the basis for the reveal-on-scroll JavaScript.

Source:

MDN Web Docs — clip-path

Maya Modeling & Rigging Tutorials
